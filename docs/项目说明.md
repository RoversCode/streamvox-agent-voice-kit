## 背景定位

`streamvox-agent-voice-kit` 的目标是把 StreamVox SDK 收敛成一层对 Agent 更友好的本地语音 Runtime。

核心分工是：

- Agent 决定“说什么”
- Runtime 决定“怎么说”

这意味着它更像是：

- 一个常驻模型会话
- 一个事件驱动的语音执行层
- 一个支持角色缓存、默认角色和能力快照的本地服务

而不是：

- 一个完整 Agent 框架
- 一个通用记忆系统
- 一个公开替代 StreamVox SDK 的全功能新 API

## 为什么现在要重构文档

- 每个模型适合什么场景
- 哪些参数值得暴露给 Agent
- 什么样的文本更容易出好效果

现在这些信息已经拆分到独立文档里，避免 README 持续膨胀，也避免多个文档各说各话。
