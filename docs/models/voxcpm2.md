# streamvox runtime 能力说明

## 支持的语言
中文、日语、韩语、印地语、泰语、越南语、印尼语、马来语、他加禄语、高棉语、老挝语、缅甸语
英语、法语、德语、西班牙语、意大利语、俄语、葡萄牙语、荷兰语、波兰语、瑞典语、丹麦语、芬兰语、挪威语、希腊语
阿拉伯语、土耳其语、希伯来语、斯瓦希里语

## 风格标签
模型支持在文本中插入少量风格语言标签，用来增强副语言表达、减少机械播报感。标签格式为 `[标签内容]`，请只使用下面这些已经明确支持的标签。

- **笑与叹息**：`[laughing]`、`[sigh]`
- **停顿与思考**：`[Uhm]`、`[Shh]`
- **疑问语气**：`[Question-ah]`、`[Question-ei]`、`[Question-en]`、`[Question-oh]`
- **情绪**：`[Surprise-wa]`、`[Surprise-yo]`、`[Dissatisfaction-hnn]`

## 注意事项
- 尽量只用上面列出的标签，不要自己扩写新的标签。
- 一句话里标签一定要少，通常 `0` 到 `1` 个就够了，不要叠很多个。
- 标签更适合短句、情绪反应、口语化表达，不适合大段客观播报。
- 像 `[laughing]` 这类常见全小写形式通常更稳。

## 基于模型能力，不同人设风格的文本示例

1. 风格：professional_assistant（专业助手）
```text
部署已经完成，日志也核对过了。接下来我继续帮你检查剩余风险。
```

2. earnest_gentle（认真乖巧）
```text
[Uhm] 我已经把结果重新核对了一遍，应该没有漏掉的问题啦。
```

3. strict_teacher（严肃教师）
```text
这个步骤不能跳过。请先把配置改正确，再继续下一步。
```

4. laid_back_expert（慵懒大佬）
```text
[sigh] 这个报错不难，我顺手帮你处理掉，你继续看结果就行。
```

5. ancient_swordsman（古风侠客）
```text
[laughing] 小小报错，何足挂齿。在下已替阁下理顺这一段脉络。
```

6. seductive_diva（妖娆御姐）
```text
[Uhm] 进度我都替你收拾妥当了，接下来想先看结果，还是先听我慢慢讲？
```

7. green_tea_girl（绿茶妹妹）
```text
[Dissatisfaction-hnn] 人家都已经帮哥哥整理好了，你怎么还不夸我一下呀。
```

8. hotheaded_bro（暴躁老哥）
```text
[Surprise-wa] 这破问题总算处理完了，兄弟你赶紧往下跑，别再卡这儿了。
```

9. extreme_chuunibyou（极端中二）
```text
[Surprise-yo] 封印已然解除，凡人，你的任务结果如今尽在本座掌握之中。
```
